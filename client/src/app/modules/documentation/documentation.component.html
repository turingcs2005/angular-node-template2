You may create multiple components under the documentation module to house markdown files for documentation.

  • High-level overview of your app
  • App workflow (You may use mermaid package for charting)
  • Main feature modules
  • Data structure
  • Transaction tables (SQL)
  • Meta data such as lookup tables (MongoDB)
  • Math (e.g. rate calculation. You may use KaTeX package to render math formulas)
